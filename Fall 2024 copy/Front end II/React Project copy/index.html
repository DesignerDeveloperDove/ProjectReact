When we would use it:

Ecpect Alot of interaction

React handles interactivity very well do to the JSX format, it allows for javascript directly in buttons/Divs/etc which can allow for easier interaction with your site

When you Need reusable code

React uses components, which are prebuilt code, it is easy to creat a component and reuse it mutiple times

when you need Fast Rendering coded Fast

React can use a series of built in functions known as "import" and "export" that allow you to creat components or even whole pages on another file and then import them in to your current page when needed. This saves time, reduces latency, and speeds up your program.

When your program needs memory

React has what are known as states, it can remember the state of a component, whether it be clicked , toggled, etc.

When creative freedom is needed

due to the JSX format mixed with components, react has been noted for being very flexible and able to produce variable types of code for varius use cases.

Why we would use it:

Rich UI

Building rich user interfaces has never been easier, react allows u=you to mix HTML & JS in its new JSX syntx, making it easier than ever to make functions like "click" work.

Custom Components

React uses what are known as "components", which are small bits of code that represent a UI aspect/feature, these can be almost anything you put your mind to, from buttons to pages, allowing for creativity.

Faster Code & Faster Rendering

React can use a series of built in functions known as "import" and "export" that allow you to creat components or even whole pages on another file and then import them in to your current page when needed. This saves time, reduces latency, and speeds up your program.

Strong Resources

Because react has worked its way up to being one of the most used libraries in dev spaces, the resources available for learning, colaborating , and overall growth of the library are extensive.

Full Stack , All in one place

React has some pretty cool extentions that can give you the capability to creat full stack apps using just react and JS. for example EXPOfor react native, allows you to dev for both IOS & Android devices. Remix alllows you tobreak apps into nested front end and back end parts and call on these parts only when needed.
