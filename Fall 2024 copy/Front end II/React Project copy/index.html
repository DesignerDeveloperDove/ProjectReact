When to use React:

When you have many interactions

React handles interactivity very well do to the JSX format, it allows for javascript directly in buttons/Divs/etc which can allow for easier interaction with your site

When reusing code

React uses components, which are prebuilt code, it is easy to creat a component and reuse it mutiple times

When you need fast rendering

React can use a series of built in functions known as "import" and "export" that allow you to creat components or even whole pages on another file and then import them in to your current page when needed. This saves time, reduces latency, and speeds up your program.

When your program needs memory

React has what are known as states, it can remember the state of a component, whether it be clicked , toggled, etc.

When creative freedom is needed

due to the JSX format mixed with components, react has been noted for being very flexible and able to produce variable types of code for varius use cases.

Why you should use React:

Rich UI

Faster Code & Faster Rendering

React can use a series of built in functions known as "import" and "export" that allow you to creat components or even whole pages on another file and then import them in to your current page when needed. This saves time, reduces latency, and speeds up your program.

Installing React

Step 1. Download

Go to Nodejs.org and download Node.


Step 2. Confirm

Open the terminal on your computer and type “node -v” and press enter to check the version of node. This will determine that the download was successful.


Step 3. Open Host

Open your files and right click on the folder you want to have react hosted on à select “new terminal at folder” or “new terminal tab at folder.”


Step 4. Check For npm

Using the terminal on local computer look for the success message (node package manager)


Step 5. IDE

Open your prefered IDE , for this ecample we'll use VS Code, and open the folder you just downloaded react on


Step 6. App.js

Open the folder titled "SRC" and from inside that folder go to the page titled "App.Js"


Step 7. Open Terminal

Once inside "App.Js" navigate to the top nav and open a new terminal in your file

Step 8. Navigating Your Files

CD is a terminal command that allows you to navigate through your folder on your local device. Use the cd command followed by the name of the folder your in (the one react was downloaded to)

Step 9. START!

Type "npm start" into the terminal to start up node.

Step 10.OPEN REACT!

Open the local file (run file) on your computer to start react
